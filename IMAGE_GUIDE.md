## 📸 Image Specifications

### **Technical Requirements:**
- **Format:** JPG (high quality) or PNG (for transparency)
- **Resolution:** Minimum 1200x800px for main images
- **File Size:** Under 500KB per image (optimized for web)
- **Quality:** Professional, high-resolution, well-lit
- **Background:** Clean, professional (white, light gray, or industrial setting)

### **Style Guidelines:**
- **Consistent:** All images should have consistent lighting and style
- **Professional:** Industrial-grade equipment photography
- **Clean:** No clutter, focus on the equipment
- **Real:** Actual product images (not renders if possible)
- **Branding:** Include subtle branding if appropriate

### **Naming Convention:**
```
{product-category}-{view-number}.jpg
Examples:
- water-purification-system-main.jpg
- ro-system-1.jpg
- sewage-treatment-plant-2.jpg
```
